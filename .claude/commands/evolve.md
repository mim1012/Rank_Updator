# 변수 진화

유전 알고리즘을 사용하여 변수 조합을 자동으로 진화시킵니다.

## 사용법

### 1. 자동 진화 (다음 세대)
```
/evolve
```

### 2. 특정 세대로 진화
```
/evolve --generation <number>
```

### 3. 진화 전략 커스터마이징
```
/evolve --elite 10 --crossover 40 --mutation 40 --random 10
```

### 4. 환경 변화 감지 후 새 탐색
```
/evolve --reset
```

## 예시

```bash
# 자동 진화 (세대 1 → 2)
/evolve

# 세대 5로 진화
/evolve --generation 5

# 커스텀 비율 (엘리트 20%, 교차 30%, 변이 40%, 랜덤 10%)
/evolve --elite 20 --crossover 30 --mutation 40 --random 10

# 환경 변화 감지 시 리셋
/evolve --reset
```

## 유전 알고리즘 구조

```
다음 세대 구성:
├── Elite (10%): 상위 성능 조합 유지
├── Crossover (40%): 엘리트 간 교차
├── Mutation (40%): 엘리트 변이
└── Random (10%): 완전 랜덤 탐색
```

## 진화 단계

1. **엘리트 선택**: 현재 세대 상위 10% 선정
2. **교차**: 2개 엘리트 조합을 섞어 새 조합 생성
3. **변이**: 엘리트 조합의 변수 1-2개 변경
4. **랜덤 탐색**: 완전 랜덤 조합 생성
5. **DB 저장**: 새 세대 저장 및 테스트 큐 등록

## 환경 변화 감지

- 최근 7일간 엘리트 성능 20% 이상 하락
- 자동으로 새로운 탐색 시작 (`/evolve --reset`)

## 주의사항

- 진화 실행 전 현재 세대 테스트 완료 확인
- 최소 10개 이상의 테스트 결과 필요
- 매일 자정 자동 실행 스케줄러 활성화 가능
