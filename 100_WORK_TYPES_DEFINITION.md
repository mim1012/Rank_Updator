## 1. 기본 탐색 (1-10)

### **work_type 1: 검색만 (트래픽 없음)**
- 동작: 검색 결과에서 상품 발견만 하고 클릭 안 함
- 소요 시간: ~4초
- 목적: 순위 체크만 수행 (베이스라인)

### **work_type 2: 상품 클릭**
- 동작: 검색 → 상품 클릭 → 즉시 종료
- 소요 시간: ~5초
- 목적: 최소 트래픽

### **work_type 3: 클릭 + 1초 체류**
- 동작: 검색 → 상품 클릭 → 1초 대기
- 소요 시간: ~6초
- 목적: 짧은 체류

### **work_type 4: 클릭 + 3초 체류**
- 동작: 검색 → 상품 클릭 → 3초 대기
- 소요 시간: ~8초
- 목적: 중간 체류

### **work_type 5: 클릭 + 5초 체류**
- 동작: 검색 → 상품 클릭 → 5초 대기
- 소요 시간: ~10초
- 목적: 긴 체류

### **work_type 6: 클릭 + 10초 체류**
- 동작: 검색 → 상품 클릭 → 10초 대기
- 소요 시간: ~15초
- 목적: 매우 긴 체류

### **work_type 7: 클릭 + 스크롤 1회**
- 동작: 검색 → 상품 클릭 → 화면 1회 스크롤
- 소요 시간: ~6초
- 목적: 기본 스크롤

### **work_type 8: 클릭 + 스크롤 3회**
- 동작: 검색 → 상품 클릭 → 화면 3회 스크롤
- 소요 시간: ~8초
- 목적: 중간 스크롤

### **work_type 9: 클릭 + 스크롤 5회**
- 동작: 검색 → 상품 클릭 → 화면 5회 스크롤
- 소요 시간: ~10초
- 목적: 많은 스크롤

### **work_type 10: 클릭 + 페이지 끝까지 스크롤**
- 동작: 검색 → 상품 클릭 → 페이지 최하단까지 스크롤
- 소요 시간: ~12초
- 목적: 전체 페이지 탐색

### 1. 요소 존재 여부 확인

일부 상품에는 특정 요소가 없을 수 있습니다:
- 동영상 (없을 수 있음)
- 옵션 (단일 상품일 수 있음)
- Q&A (없을 수 있음)
- 쿠폰 (없을 수 있음)

**대응 방안**:
```typescript
async function executeWorkType(workType: number, page: Page) {
  try {
    switch (workType) {
      case 58: // 쿠폰 받기
        const couponButton = await page.$('.coupon-button');
        if (couponButton) {
          await couponButton.click();
        } else {
          console.log('쿠폰 없음, 대체 동작 수행');
          // 대체: 가격 정보 확인
          await page.evaluate(() => window.scrollTo(0, 200));
        }
        break;
    }
  } catch (error) {
    console.error(`WorkType ${workType} 실행 실패:`, error);
  }
}
```
